# 🌦️ Weather Hub

A simple and modern **React Weather App** built with **Vite**, **Axios**, and **TailwindCSS**.  
It fetches real-time weather data from the **OpenWeatherMap API** and displays temperature, humidity, and wind speed for any city.

---

## 🚀 Features
- 🔍 Search weather by city name
- 🌡️ Displays **temperature** in Celsius
- 💧 Shows **humidity percentage**
- 🌬️ Displays **wind speed**
- 🎨 Styled with **TailwindCSS** for a clean UI
- ⚡ Built with **Vite** for fast development

---

## 🛠️ Tech Stack
- **React** – UI framework
- **Vite** – Development environment
- **Axios** – API requests
- **TailwindCSS** – Styling
- **React Icons** – Weather-related icons
